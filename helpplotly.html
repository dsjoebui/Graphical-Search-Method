Guidelines for the Graphical Search Method

About

This interactive graph is used as a way to filter and select scientific publications. This project was inspired by the time consuming task of literature review. This is propogated through the current search method of a typical list view when searching for papers, which can be daunting when going through thousands of papers. To present the data in a condensed, organized manner, the graph was set up to sort papers by the number of times a paper is cited and by the relevance to keywords or articles input by the user. The goal is to ultimately provide researchers with a way to more efficiently screen papers and possibly provide an insight on the metadata. The data used in this graph was webscraped from PubMed, using the search term, lung cancer. As a disclaimer, this project is intended only as an academic capstone-project and not for any commercial use.

Search

The search function takes the user input of any words or sentences and computes the revelance of each publication title to the input. This function can be used in conjunction with the 'Selected Article' function. The default term is 'lung cancer', which is set when the input area is blank.

Advanced Search

The advanced search function takes the user input of words and filters the publication titles based on the syntax used.


Syntax

inputword -- User input with no syntax will only show publication titles with any instance of the input. For example: the user input, receptor, will show the publication titles, 'the ace2-receptor' or 'epithelial cell receptors'.

inputword1, inputword2, inputword3 -- User input with a comma between words will only show publication titles with instances of all the input words. For example: the user input, statistics, cancer, will show the publication title, 'statistics of tumour growth from lung cancer, 2019'.

!:inputword -- User input beginning with !: will only show publication titles NOT containing the input.

E:inputword -- User input beginning with E: will only show publication titles containing the exact input as a standalone word. For example: user input, receptor will show the publication title, 'the ace2 receptor', but will NOT show 'the ace2-receptor' or 'ace2 receptors'.

!E:inputword -- User input beginning with !E: will only show publication titles NOT containing the exact input as a standalone word.

Selected Article

The selected article title will be used to the compute the relevance of the selected article to all other articles. This can be used in conjunction to the search function. Only one article can be selected to be used for computing relevance. No article selected is the default and is not accounted for when calculating relevance.

Date Range

The date's range from 2011 to August 2021. The dates represent the date that the articles were published. Note: the number of papers shown heavily rely on the date range filter and will affect the computational time need to create the graph.

Filters

Articles containing a property of the checked boxes will be shown. The label filters checkbox creates a legend and labels the articles by markercolor and markershape based on the selected properties.

Number of Citations

The number of citations refer to the number of times an article is cited. Articles within the slider range will be shown.

How to use the Graph

The graph is made using Plotly. When hovering over the plot, a toolbar will appear on the top right for interactive functions involving zooming, selection, and panning. Data can be selected by individually clicking on the data points or using the box selection or lasso selection tool from the toolbar. When data points are selected, a table below the graph will be populated with the title and abstract of the selected points.

Selected Data

After selecting publications from the graph, a table below the graph will be filled with the title and abstract of the selected publications. Below the headers is a filter cell, where the cells can be filtered based on user input. See https://dash.plotly.com/datatable/filtering, under the filtering options to see syntax.

Save Selected Data

The 'save selected data' button is located at the bottom of the webpage and will become available only after selection of data. When the button is clicked, the selected data will be downloaded into a csv file. The data contains the information: index, title, abstract, date, publicationtype, free or paid article, journal, number of times cited, doi and URL link.
